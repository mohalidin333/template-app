# Essential Next.js SEO Setup (All You Need)

## 📦 Install Only This
```bash
npm install next-sitemap
```

## 🗂️ Simple Structure (Only 4 Files to Edit)
```
your-project/
├── next-sitemap.config.js          ← CREATE THIS
├── src/app/
│   ├── layout.js                   ← EDIT THIS
│   └── page.js                     ← EDIT THIS
└── package.json                    ← ADD ONE LINE
```

## 1. `package.json` (Add one line)
```json
{
  "scripts": {
    "postbuild": "next-sitemap"
  }
}
```

## 2. `next-sitemap.config.js` (Basic version)
```javascript
module.exports = {
  siteUrl: 'https://your-domain.com',
  generateRobotsTxt: true,
  exclude: ['/admin/*', '/api/*'],
}
```

## 3. `src/app/layout.js` (Minimal metadata)
```javascript
export const metadata = {
  title: {
    template: '%s | Your Site Name',
    default: 'Your Site Name - Your tagline',
  },
  description: 'Your site description in 150-160 characters',
  openGraph: {
    title: 'Your Site Name',
    description: 'Your site description',
    url: 'https://your-domain.com',
    siteName: 'Your Site Name',
    images: ['/og-image.jpg'],
    locale: 'en_US',
    type: 'website',
  },
  twitter: {
    card: 'summary_large_image',
    title: 'Your Site Name',
    description: 'Your site description',
    images: ['/og-image.jpg'],
  },
}

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

## 4. `src/app/page.js` (Homepage)
```javascript
export const metadata = {
  title: 'Home',
  description: 'Specific description for homepage',
}

export default function Home() {
  return <h1>Welcome</h1>
}
```

## 5. For Other Pages (Example: About)
```javascript
// src/app/about/page.js
export const metadata = {
  title: 'About Us',
  description: 'Learn about our company and mission',
}

export default function About() {
  return <h1>About Us</h1>
}
```

## 🎯 This Setup Automatically Gives You:
- ✅ **Page titles** for Google search results
- ✅ **Meta descriptions** for search snippets
- ✅ **Open Graph** tags (Facebook, LinkedIn sharing)
- ✅ **Twitter Cards** for Twitter sharing
- ✅ **Sitemap.xml** (automatically generated)
- ✅ **Robots.txt** (automatically generated)
- ✅ **Canonical URLs** (Next.js handles this)
- ✅ **Mobile-friendly** viewport (Next.js default)
- ✅ **Fast loading** (Next.js optimizations)

## 🚀 Deploy Steps
1. `npm run build` - Builds your site + generates sitemap
2. Deploy to your hosting platform
3. Submit sitemap to Google Search Console

## 💡 That's It!
This covers **90% of SEO benefits** with minimal complexity. Add more features only when you actually need them.
